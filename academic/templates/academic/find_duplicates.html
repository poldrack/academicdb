About Duplicate Detection
How it works:
  • Compares publication titles after normalizing case, punctuation, and common words
  • Groups publications with very similar titles together
  • Only includes non-ignored publications in the analysis
Common scenarios:
  • Preprint + Journal Article: Same work published as preprint then in journal
  • True Duplicates: Same publication added multiple times from different sources
  • Similar Titles: Different publications with coincidentally similar titles
Recommended actions:
  • Mark as Ignored: For preprints when you have the journal version
  • Delete: For true duplicates or unwanted entries
  • Keep Both: If they are actually different publications
